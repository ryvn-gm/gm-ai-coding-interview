# 歡迎來到 Signal / Algorithm Engineer 面試！

Hi，很高興你來到這一關！

這封信是幫你提前了解面試流程和需要準備的東西，讓你當天可以專注在解題上。

## 面試概覽

| 項目 | 說明 |
|------|------|
| 形式 | Live Coding + 口頭設計 |
| 總時間 | 約 90 分鐘 |
| 語言 | Python |

## 流程

面試分成三個部分，中間不會休息但節奏不會太趕：

1. **Task 1（約 30 分鐘）** — 在一個現有的 Python codebase 上加入新功能
2. **Task 2（約 40 分鐘）** — 分析一份資料並驗證產品結論
3. **Task 3（約 15 分鐘）** — 口頭系統設計討論，不用寫 code

每個 task 結束後會有幾分鐘的問答時間。

## 需要準備的東西

- **你自己的電腦**，可以跑 Python 的環境就好
- **Python 套件**：請先確認有安裝 `numpy`、`scipy`、`pandas`（如果沒有，面試前裝好就行）
- **你習慣用的 editor / IDE**，用什麼都可以

## 關於 AI 工具

**全程開放使用任何 AI 工具，沒有限制。**

ChatGPT、Claude、Copilot、Cursor，想用什麼都可以。我們在意的不是你有沒有用 AI。

## 最後

如果有任何問題，隨時問我們。面試當天也一樣 — 有任何不清楚的地方直接問，這不會扣分。

期待跟你聊！
