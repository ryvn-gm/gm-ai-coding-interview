# Task 2 — 資料分析與驗證

## 背景

我們的智慧戒指會記錄用戶的每日活動量和睡眠品質。這份資料是從後台匯出的 12 位用戶、7 天的紀錄。

## 任務

PM 說初步分析顯示「運動量越高，當晚睡眠品質越好」，想做成產品功能。
請你驗證這個結論，然後告訴面試官你的發現。

## 資料說明

檔案：`sleep_exercise.csv`

| 欄位 | 型別 | 說明 |
|------|------|------|
| `user_id` | string | 用戶 ID（U001 ~ U012） |
| `date` | date | 紀錄日期（2024-01-08 ~ 2024-01-14） |
| `steps` | int | 當日步數，來自戒指的加速度計原始累計 |
| `active_minutes` | int | 當日活動時間（分鐘），由演算法從加速度資料判定 |
| `sleep_score` | float | 睡眠品質分數（0–100），戒指在用戶醒來時自動量測並記錄到該天 |

共 84 筆（12 人 × 7 天）。

## 環境

- 語言：Python
- 可用套件：pandas、numpy、scipy、matplotlib，以及任何你覺得需要的
- 全程開放使用任何 AI 工具，沒有限制
